A new version - what did change?

OS 2.7 represents one of the biggest changes and improvements ever made to the Virus TI series software. It also represents nine month of work and the feedback of an extensive public beta test program.

Among hundreds of refinements and bug fixes, the following new features have been added:

- A new effect Glitch/FSU effect called Atomizer
- An optional third USB stereo output sending audio from the TI hardware into Virus Control
- Support for multiple Viruses on one computer
- Virus Control Center application

Please take the time to familiarise yourself with the new additions. Please read through the documentation supplied, especially the sequencer tutorials which were improved for your convenience. Yes, we know that nobody likes reading support documents but without knowing about the new features you might miss something which might inspire you to write greater music.

A dedicated addendum explains all additions in-depth and can be found using the button at the right lower corner of this window.
